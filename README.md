# CollabHub: A Web-Based Peer Tutoring and Academic Support System

CollabHub is a collaborative learning platform designed to connect students and teachers in an interactive academic environment. The system enables users to ask and answer academic questions, share knowledge, and access reliable educational support through a centralized and searchable platform.

### Features

#### User Roles

* **Student**

  * Ask questions
  * Upload images with questions
  * Answer questions from other students

* **Teacher**

  * Answer student questions
  * Receive a **Teacher Verified** badge on responses to indicate authority and credibility

#### Q&A System

* Post academic questions with optional image attachments
* Community-driven question answering
* Teacher-verified responses for trusted academic guidance
* Previously uploaded questions remain accessible to all users, reducing duplicate submissions

#### Search and Tagging

* Organize questions using hashtags such as `#programming`, `#math`, and `#biology`
* Search questions by keywords or topic tags
* Support for multiple tags per question for easier content discovery

#### Home Feed

* Displays unanswered questions to encourage participation
* Allows users to answer questions directly from the feed
* Enables quick posting of new questions

### Purpose

CollabHub promotes collaborative learning by providing a space where students can seek help, share knowledge, and receive verified academic support from teachers. The platform aims to make educational assistance more accessible, organized, and efficient.
